# Stratégie — Mikael El Ouazzani (AZMK)

## Funnel actuel

```
Meta Ads
  └─► Page d'optin (inscription webinaire)
        └─► Groupe WhatsApp + séquence email + vidéos d'endoctrinement (J-7 → J)
              └─► Webinaire live (dimanche ~12h ou 18h)
                    └─► Appel de vente (SDR → Closer)
                          └─► Vente du programme
```

## Détail des étapes

### 1. Ads (Meta)
- Objectif : trafic vers page optin
- Angles testés :
  - "Comment obtenir 15 à 25 entretiens qualifiés par semaine"
  - "Le système qui génère +10 opportunités par semaine en automatique"
  - "Ce que nos ex-recruteurs ESN ont appris en plaçant des centaines de freelances"
- Creative strategist + media buyer gérés par SAFE

### 2. Page d'optin
- Promesse principale : accéder au système data-driven pour ne plus jamais subir l'intercontrat
- Formulaire : prénom + email + téléphone (facultatif)
- CTA : "Réserver ma place — 100% gratuit"

### 3. Séquence de show-up (J-7 → J)

| Canal | Volume |
|---|---|
| Emails | 7 à 9 (éducatifs + teasers + émotionnels) |
| WhatsApp | 7 à 8 messages (texte, vidéo, vocal) |
| SMS | 1 confirmation + 1 rappel avant live |
| Vidéos d'endoctrinement | 2 (email + WhatsApp) |

**Objectifs des vidéos :**
- Vidéo 1 : casser la croyance "le freelancing = CDI mieux payé" → posture entrepreneuriale
- Vidéo 2 : casser la croyance "c'est la faute du marché saturé" → erreurs des freelances

### 4. Webinaire
- Format : live (~1h30) + Q&A
- Cadence : chaque dimanche
- Structure : pas encore formalisée dans les docs (à documenter)
- Outil : WebinarJam / n8n pour les liens

### 5. Appel de vente
- SDR + Closer en place
- Score → gamifier → valider motivation + problématique + point de départ
- **Problème identifié :** leads qualifiés mais closing faible (20%)

## Priorité stratégique actuelle

**Problème #1 = les sales.** Les leads entrants sont qualifiés. Le goulot d'étranglement est dans la conversion appel → vente. Deux axes à investiguer :
1. Transcripts des calls (CRM Airtable) → identifier les objections récurrentes
2. Script de vente et posture du closer → à auditer
